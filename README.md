--> About This Project
Built an interactive Excel dashboard to analyze Adidas US sales data 9,648 transactions across 6 major retailers, 5 US regions, and 3 sales channels. The goal was to uncover which products, regions, and channels drive the most revenue and profit.

--> Dataset
Source: [Adidas US Sales Dataset Kagg](https://www.kaggle.com/datasets/heemalichaudhari/adidas-sales-dataset)
Covers sales from January 2020 to December 2021. Includes retailer info, product categories, pricing, units sold, operating profit, and sales method (in-store, online, outlet).

--> What I Did
- Cleaned raw data: removed extra header rows, fixed date formats, added calculated columns (Year, Month, Quarter)
- Converted data to Excel Table format for easier pivot table creation
- Built 6 Pivot Tables covering sales by region, product, retailer, month, state, and sales method
- Designed a dashboard sheet with KPI cards, 6 charts, and interactive slicers
- Connected all pivot tables through the Data Model so slicers filter everything at once

--> Key Numbers
- Total Revenue: $899.9M | Total Profit: $332.1M | Units Sold: 2.48M | Avg Margin: 42.3%
- New York leads all states at $64.2M revenue
- Men's Street Footwear is the top-selling product category

--> Tools
Microsoft Excel — Pivot Tables, Charts, Slicers, Data Model, Conditional Formatting
